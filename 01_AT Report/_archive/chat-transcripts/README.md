# Archived chat-transcript sources

These are the raw AI-chat exports (mostly Korean, some English, several duplicate "Branch ·" conversation forks) this ICM workspace was built from. They're **superseded**, not deleted — kept for provenance and for the content that hasn't been triaged into `references/` or `records/` yet (see each case's `CONTEXT.md` "Outstanding" section for pointers back into specific files here).

Don't treat anything in here as authoritative going forward — where content from these files was extracted, it was translated, cleaned, and re-homed under `../../references/` (reusable/stable) or `../../records/<case>/` (case-specific). Where extraction found unreconciled conflicts between drafts, that's noted explicitly in the destination file rather than silently resolved — check the destination file's frontmatter/notes before assuming this archive still has something new to add.

`../../markdown_files/PDFs/` (rendered PDF exports of these same files) and `../../markdown_files/Archive/` (an unrelated personal note) were left in place and are not part of this workspace.

`Original Text files.zip` is the same 19 transcripts as plain `.txt` (confirmed by diff against `Bioness L300 Go 보고서.md` — identical, 2370 lines both). It's a format duplicate, not additional source material — don't re-mine it expecting new content.
