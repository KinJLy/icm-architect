# 04_washout — record deterioration after the device is returned

One job: prove the trial's gains were caused by the device, not incidental.

## Inputs
- Working (this run): ../03_community-trial/output/outcome-measures.md (end-of-trial baseline to compare against)
- Reference (every run): ../../../references/community-trial-protocol.md (washout logbook template)

## Process
1. On device return, start the 3-week withdrawal logbook (daily + weekly summary format in the protocol reference).
2. Re-run the same outcome measures used at end-of-trial.
3. Write up: trial (improvement) + withdrawal (deterioration) = clinical necessity, not convenience. This comparison is the single strongest piece of evidence in a High Cost AT report — don't bury it.

## Outputs
- washout-findings.md → output/

## Human check
Confirm the deterioration is documented with the same rigour as the improvement — assessors will look for this contrast specifically.
