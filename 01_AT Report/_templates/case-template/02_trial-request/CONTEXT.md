# 02_trial-request — request funding for a community trial

One job: produce a signed-ready support letter requesting NDIS funding for a device trial.

## Inputs
- Working (this run): ../01_intake-assessment/output/assessment-findings.md
- Reference (every run): ../../../references/support-letter-templates.md
- Reference (every run): ../../../references/org-info.md

## Process
1. Pick the template variant matching who's authoring it (orthotist / physiotherapist / generic) from `support-letter-templates.md`.
2. Fill in participant details, current supports, functional impact (cover all relevant NDIS domains — mobility, self-care, self-management, participation), and why a trial (not straight purchase) is the right next step.
3. State explicitly: "reasonable and necessary."

## Outputs
- support-letter.md → output/

## Human check
Author signs before lodging. Nothing in this letter should overstate what's in `assessment-findings.md`.
