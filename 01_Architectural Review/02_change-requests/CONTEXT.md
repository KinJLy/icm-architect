# 02_change-requests — draft the ask back to Masterton

One job: turn this round's comparison table into a written list of changes to request from Masterton.

## Inputs
- Working (this round): `../01_compare/output/round-NN-comparison.md` (latest round)
- Reference (every round): `../_reference/voice.md`

Do NOT load: the raw plan files or Pajic costing — the comparison table already carries what matters.

## Process
1. Read the latest comparison table.
2. For each item you want changed, state it plainly: what to change, and back to what/why.
3. Leave out anything you're accepting as-is.
4. Keep it short enough to send as-is — no fluff, per `voice.md`.

## Outputs
- `round-NN-change-requests.md` → `output/` (same NN as the comparison it's drawn from)

## Human check
Read it as if you were Masterton receiving it — is every ask unambiguous? Edit in place before sending.
